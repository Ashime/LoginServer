/help
Brings up all of the command options.

/activity
Displays stats of server activity.

/font #
Changes the font size. If # is 0 then server will set the font to size
12.

/info
Displays all server info.

/maintenance true/false
Not avaliable yet.

/restart # 
Restarts the nioserver. If # is 0 then instant restart, otherwise there
is restart delay.
